What Happens Automatically

Ultrapilot - Parallel Autopilot

Say "ultrapilot" or "parallel build" for 3-5x faster execution with up to 5 concurrent workers handling different parts of your project.

Swarm - Coordinated Agents

Spawn N agents with shared task pool. Each agent claims atomic tasks, executes them, and marks completion with 5-minute timeouts.

Pipeline - Sequential Chaining

Chain agents in sequence with data passing between stages. Use built-in presets (review, implement, debug, refactor) or create custom chains.

Ecomode - Token-Efficient Parallelism

Say "eco" for budget-friendly parallel execution. Like ultrawork but with smaller agents and tighter budget controls for Pro plan users.

Complex Tasks - Parallelized

Give me a complex task and I automatically parallelize work across specialist agents for maximum speed.

Planning - Interview Mode

Say "plan this" and I start an interview to understand your requirements before creating a comprehensive plan.

Must Complete - Persistence

Say "don't stop until done" and I persist until the task is verified complete. No partial implementations.

UI Work - Design Sensibility

When working on UI/frontend, I automatically activate design expertise for better results.

Magic Keywords

Power users can use these anywhere in messages for explicit control
